# Log History

All your debug logs are saved and organized for easy access.

## Features

- **Persistent Storage** - Logs saved to `reports/` folder
- **Auto-Retention** - Configurable max files (default: 10)
- **File Splitting** - Large logs can split automatically
- **Metadata** - File size, line count, duration shown

## Right-Click Actions

- **Rename Log** - Give logs meaningful names
- **Tag Log** - Add comma-separated tags (#bug, #feature)
- **Export HTML** - Share styled logs with teammates
- **Copy Deep Link** - URL that opens exact log + line

## Searching

Use **Saropa Log Capture: Search Log Files** (Ctrl+Shift+P) to search across all saved logs.

# Keyboard Shortcuts

Master these shortcuts to work faster with your logs.

## Navigation

| Key | Action |
|-----|--------|
| **Space** | Pause/resume capture |
| **Home** | Jump to start |
| **End** | Jump to end |

## Search

| Key | Action |
|-----|--------|
| **Ctrl+F** | Open search bar |
| **F3** | Next match |
| **Shift+F3** | Previous match |
| **Escape** | Close search |

## Actions

| Key | Action |
|-----|--------|
| **M** | Insert timestamp marker |
| **P** | Pin/unpin current line |
| **N** | Add note to current line |
| **W** | Toggle word wrap |
| **A** | Toggle app-only stack frames |

## Copy

| Key | Action |
|-----|--------|
| **Shift+Click** | Select multiple lines |
| **Ctrl+C** | Copy as plain text |
| **Ctrl+Shift+C** | Copy as markdown |

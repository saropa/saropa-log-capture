# Share Investigation

**To open a shared .slc file:** In the Investigation panel click **Open .slc file** (or Command Palette → **Import .slc Bundle**), then pick the .slc file in the file dialog. Works for investigations and session bundles.

**Share options:** Gist, Export as .slc, Copy deep link (local file), LAN, Upload URL, Shared folder. Recipients: Ctrl+Shift+P → type **Import .slc Bundle** → pick the file; or click a `vscode://` link.

**Gists:** Secret gists don’t expire. To delete: open the gist on GitHub → Delete (trash icon).
